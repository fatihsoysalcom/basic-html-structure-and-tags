# Basic HTML Structure and Tags

This example demonstrates the fundamental structure of an HTML document, including the <head> and <body> sections. It showcases common HTML tags like heading tags (<h1> through <h6>), paragraph tags (<p>), and anchor tags (<a>) for creating hyperlinks. The code includes inline comments explaining the purpose of each section and tag.

## Language

`html`

## How to Run

Save the code as `index.html`.
Open the `index.html` file in any web browser.
